# GPA Calculator
A simple GPA calculator program written in C. This program allows users to input grades for multiple classes and calculates their GPA.

## Features
- Input number of classes
- Enter grade points (0.0 - 4.0)
- Input validation for grades
- Calculates and displays GPA

## How to Run
gcc gpa_calculator.c -o gpa_calculator 
./gpa_calculator
